# portfolio-app

A modern personal portfolio built with React, TypeScript, and Vite.

## Prerequisites

- Node.js 18+
- npm 9+

## Setup

```bash
git clone git@github.com:Offthearc/portfolio-app.git
cd portfolio-app
npm install
```

## Run dev server

```bash
npm run dev
```

Opens at `http://localhost:5173`

## Build

```bash
npm run build
```

## Tests

```bash
npm test -- --run
```

## Lint

```bash
npm run lint
```
